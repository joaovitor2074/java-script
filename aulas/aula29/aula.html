Estudo sobre Arrays em JavaScript

Arrays são estruturas de dados que permitem armazenar múltiplos valores em uma única variável.

Exemplo de criação de um Array

        
            // Criando um array
            var frutas = ["Maçã", "Banana", "Laranja"];
            console.log(frutas);

Acessando elementos do Array

        
            // Acessando o primeiro elemento do array
            console.log(frutas[0]); // Maçã

Adicionando elementos ao Array

        
            // Adicionando um novo elemento ao array
            frutas.push("Uva");
            console.log(frutas);

Removendo elementos do Array

        
            // Removendo o último elemento do array
            frutas.pop();
            console.log(frutas);
